# CoralPure (珊瑚去水印) - AI 协作开发指南

## 🤖 项目概述
**中文名**：珊瑚去水印  
**英文名**：CoralPure  
**技术栈**：微信小程序 + Node.js/Python + FFmpeg/OpenCV  
**核心功能**：解析主流平台（抖音/快手/B站/TikTok）短视频链接，去除水印并提供下载
